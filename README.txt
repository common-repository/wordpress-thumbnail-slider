== Installation ==

1. Download the zipped plugin file to your local machine.
2. Unzip the file.
3. Upload the "wpns_thumbnail_slider" folder to the "/wp-content/plugins/" directory. (DO NOT CHANGE ANY FILE OR FOLDER NAMES)
4. Activate the plugin through the "Plugins" menu in WordPress.
5. Add the folowing code anywhere within your theme where you want the content gallery to be displayed: `<?php if(function_exists('wpns_thumbnail_slider')) { wpns_thumbnail_slider(); } ?>`
6. Make Sure to change file permission of the 'cache' folder to 777 it is inside the "wpns_thumbnail_slider" folder.
